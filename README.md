# PixelMplusフォントMod
* 日本語化用フォントをPixelMplusに差し替えただけのModです

# 使用方法
* 各自が使用しているmodloaderにて反映してください。

# 利用責任
* 本ソフトウェアを利用したことにより発生した、いかなる損害や問題について、当方および元となるフォントファイル作成者はその責を負いません。

# ファイル構成
* c&c.ttf > PixelMplus10-Regular.ttf
* c&cnew.ttf > PixelMplus10-Regular.ttf
* JustinFont10.ttf > PixelMplus10-Regular.ttf
* JustinFont11.ttf > PixelMplus10-Regular.ttf
* JustinFont11Bold.ttf > PixelMplus10-Bold.ttf
* JustinFont12.ttf > PixelMplus12-Regular.ttf
* JustinFont12Bold.ttf > PixelMplus12-Bold.ttf
* JustinFont7.ttf > PixelMplus10-Regular.ttf
* JustinFont8.ttf > PixelMplus10-Regular.ttf
* num_font.ttf > FTLオリジナル

 *  JIS X 0213⇔Unicodeの文字コード変換表は以下を使用させていただきました。
   *  JIS X 0213:2004 漢字8ビット符号とUnicodeの対応表 http://x0213.org/codetable/jisx0213-2004-8bit-std.txt
